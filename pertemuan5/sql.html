SQL, DDL, DML, DCL dan TCL

Apa itu SQL?

SQL (Structured Query Language) adalah bahasa yang digunakan untuk mengelola dan memanipulasi basis data relasional. SQL digunakan untuk melakukan berbagai operasi pada data, seperti mengambil, menyisipkan, memperbarui, dan menghapus data dalam tabel.

SQL terdiri dari beberapa subbahasa, termasuk DDL, DML, DCL, dan TCL, yang masing-masing memiliki fungsi dan tujuan yang berbeda dalam pengelolaan basis data.

Subbahasa SQL Fungsi Contoh Perintah
DDL (Data Definition Language) Mendefinisikan struktur database, seperti pembuatan dan pengubahan tabel. CREATE TABLE, ALTER TABLE, DROP TABLE
DML (Data Manipulation Language) Memanipulasi data dalam tabel, seperti mengambil, menyisipkan, memperbarui, dan menghapus data. SELECT, INSERT, UPDATE, DELETE
DCL (Data Control Language) Mengontrol akses pengguna terhadap data dalam database. GRANT, REVOKE
TCL (Transaction Control Language) Mengelola transaksi dalam database. COMMIT, ROLLBACK

DDL (Data Definition Language)

DDL adalah subset dari SQL yang digunakan untuk mendefinisikan struktur database, termasuk pembuatan, pengubahan, dan penghapusan tabel serta skema. Contoh perintah DDL meliputi:

  • CREATE TABLE: Membuat tabel baru.
  • ALTER TABLE: Mengubah struktur tabel yang sudah ada.
  • DROP TABLE: Menghapus tabel dari database.

DML (Data Manipulation Language)

DML adalah subset dari SQL yang digunakan untuk memanipulasi data dalam tabel. Contoh perintah DML meliputi:

  • SELECT: Mengambil data dari tabel.
  • INSERT: Menyisipkan data baru ke dalam tabel.
  • UPDATE: Memperbarui data yang sudah ada dalam tabel.
  • DELETE: Menghapus data dari tabel.

DCL (Data Control Language)

DCL adalah subset dari SQL yang digunakan untuk mengontrol akses pengguna terhadap data dalam database. Contoh perintah DCL meliputi:

  • GRANT: Memberikan hak akses kepada pengguna.
  • REVOKE: Mencabut hak akses dari pengguna.

TCL (Transaction Control Language)

TCL adalah subset dari SQL yang digunakan untuk mengelola transaksi dalam database. Contoh perintah TCL meliputi:

  • COMMIT: Menyimpan perubahan yang telah dilakukan dalam transaksi.
  • ROLLBACK: Membatalkan perubahan yang telah dilakukan dalam transaksi.
